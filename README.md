# Rogue
Small simple game in development.


## Controls

- WASD -- Move
- F -- Interact
- Tab -- Inventory
- +/- -- zoom
- B -- Builder Mode
- 1..9 / Mouse Wheel -- Change Inventory slots
- LMB -- Attack/BuildStructure

### Ui controls 

- LMB -- Select, take stack
- Shift + LMB -- Take half-stack
- Ctrl + LMB -- Take one
- Backquote -- turn console
