# MVP Roadmap - Top-Down Roguelike

Цей документ підсумовує поточний стан гри та план дій для мінімально життєздатної версії (MVP).

---

## 1. Ціль гри

* Основна мета: виживання.
* Опціонально: вбивство останнього боса, але для MVP не обов'язково.
* Гравець досліджує печери, добуває ресурси, крафтить предмети та бореться з ворогами.

---

## 2. Ресурси

| Ресурс     | Призначення | Використання                                              |
| ---------- | ----------- | --------------------------------------------------------- |
| Structorix | Структурний | Держаки, стіни, каркас для станків/печей                  |
| Inferium   | Магічний    | Крафт спелів або їх компонентів                           |
| Sectura    | Зброя       | Основна зброя: мечі, сокири, покращена кирка              |
| Mechanae   | Технічний   | У комбінації з Structorix → станки, печі, апгрейди табору |

---

## 3. Структури табору

| Структура | Ресурси               | Призначення                                                    |
| --------- | --------------------- | -------------------------------------------------------------- |
| Верстак   | Structorix            | Крафт зброї та інструментів                                    |

---

## 4. Зброя (MVP)

| Зброя           | Ресурси              | Призначення                                       |
| --------------- | -------------------- | ------------------------------------------------- |
| Базова кирка    | Невідомий ресурс     | Початкова зброя/інструмент для добування ресурсів |
| Сокира          | Sectura + Structorix | Інструмент і зброя ближнього бою                  |
| Меч             | Sectura + Structorix | Основна зброя ближнього бою                       |
| Покращена кирка | Sectura + Structorix | Покращена ефективність майнінгу та бою            |

---

## 5. Основні завдання для MVP

1. **Збереження та відновлення стану гри**

   * Серіалізація стану гравця і світу.
   * Функція завантаження гри.

2. **Спавнери ворогів**

   * Базові спавнери для небезпечної зони.
   * Базовий спавн для бос-зони (опціонально).

3. **Візуал**

   * Базові текстури/спрайти для гравця, ворогів, ресурсів, структур.
   * Прості анімації ударів або взаємодії.

4. **HUD / UI**

   * Життя гравця.
   * Лічильник Inferium.
   * Повідомлення про підняття ресурсів або крафт.

5. **Головне меню (мінімальне)**

   * Start / Continue / Exit.

6. **Вороги**

   * Задизайнити двох ворогів: базовий і більш складний.
   * Придумати зовнішній вигляд, базову поведінку та шкоду.

7. **Додаткові пункти (опціонально)**

   * Звукові ефекти (удари, смерть, крафт).
   * Мінімальні ефекти взаємодії з ресурсами.
   * Індикація предметів на землі.

---

Цей roadmap можна використовувати як чек-лист для створення MVP, від найважливішого до додаткових поліпшень.
