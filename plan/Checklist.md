# MVP Starter Checklist - Top-Down Roguelike

Цей чек-лист допомагає розставити пріоритети для початку роботи над MVP та відслідковувати прогрес.

---

## **Пріоритети для старту MVP**

* [ ] **Базова механіка гравця**

  * Рух, атака, базова кирка для добування ресурсів.
  * Життя гравця та отримання шкоди.

* [ ] **Ресурси та генерація печер**

  * Генератор карт із випадковими кімнатами/тунелями.
  * Розташування ресурсів: Structorix, Sectura, Mechanae, Inferium.

* [ ] **Крафт і табір**

  * Верстак, піч, сундук.
  * Крафт базових предметів: меч, сокира, покращена кирка.

* [ ] **Вороги та спавнери**

  * Два базових вороги для небезпечної зони.
  * Спавнери для цих ворогів.

* [ ] **HUD / UI**

  * Життя гравця.
  * Лічильник Inferium.
  * Повідомлення про підняття ресурсів або крафт.

* [ ] **Головне меню (мінімальне)**

  * Start / Continue / Exit.

* [ ] **Додатково (після базових систем)**

  * Звукові ефекти (удари, смерть, крафт).
  * Мінімальні ефекти взаємодії з ресурсами.
  * Індикація предметів на землі.
