# About the assets

## Style 
The game currently doesn't have a fully established visual style yet.  

### In-game atmosphere
The entire game takes place inside a massive,  
seemingly endless cave system.
The overall atmosphere is dark,   
isolated and cave-focused.  
Terraria cave environments are a good reference  
for the general mood and exploration feeling.

### Art
I want to keep the art simple and minimalistic.  
For now everything is in APOLO color palette, but I  
don't mind switching to another one, if it looks better.

### Size

I currently use 32x32 px tiles for blocks.  
For other sprites it can be different.

### Assets

Many current assets are temporary placeholders  
created mainly for gameplay implementation.   
I'm fully open to redesigning existing assets and  
introducing new visual ideas if they fit the game's  
direction.

## Important goals

- Readability is more important than detail
- Assets should work well in dark environments
- Strong silhouettes are preferred
- Minimal visual noise
- Consistent scale between objects
